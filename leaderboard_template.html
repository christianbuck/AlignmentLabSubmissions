MTM 2013 - Alignment Lab Leaderboard

    How to get on the list:

    1. Come up with a great alignment algorithm and implement it.
    2. Get a github account and send your github username to cbuck@lantis.de
       Wait to be added to the collaborators
    3. Clone the submission repository:
       $ git clone git@github.com:christianbuck/AlignmentLabSubmissions.git
    4. Add your alignment in a folder of your name
       $ cd AlignmentLabSubmissions
       $ cd submissions
       $ mkdir YOURUSERNAME
       $ cd YOURUSERNAME
       $ cp /PATH/TO/YOURALIGNMENT alignment
       $ git add alignment
       $ git commit alignment
       $ git push
    5. Wait for up to 5 minutes until the script ran and added you
    6. Profit!
